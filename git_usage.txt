# Create a new repository on the command line.
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/RikuRedfox/coding_session.git
git push -u origin main

or Push an exiting repository from the command line.
git remote add origin https://github.com/RikuRedfox/coding_session.git
git branch -M main
git push -u origin main

other : 
git status // get status of the working tree if there is any changes
