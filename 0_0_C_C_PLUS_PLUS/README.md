An Introduction to C++

A general-purpose programming language.
Desgined and implemented by Bjarne Stroustrup in 1979 at Bell Labs.
It runs on variety of platforms, such as Windows, MacOS, and the various version of UNIX.
Fun Fact: The initial version of C++ was called "C with classes."

C++ : An Extension of C
C++ is:
- a better C.
- supports data abstraction.
- supports object-oriented programming.
- supports generic programming
